
# TorrServer GitHub Pages Finder

Сайт работает **только через GitHub**.

## Что делает

✔ GitHub Action сканирует интернет  
✔ ищет TorrServer серверы  
✔ сохраняет 10–20 рабочих IP  
✔ сайт показывает их без http

Формат:

IP:PORT

пример:

91.193.43.141:8090

---

# Как запустить

1. Создай репозиторий на GitHub

2. Загрузи все файлы из этого архива

3. Включи GitHub Pages

Settings → Pages → Deploy from branch → main

4. Открой сайт

https://USERNAME.github.io/REPOSITORY

5. Нажми кнопку **Обновить**

---

# Автоматическое обновление

GitHub будет искать новые IP каждые **30 минут**.
